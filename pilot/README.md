# VALOUR Pilot Workspace

This folder contains the live pilot operating structure for VALOUR.

The `docs/` folder remains the product, architecture, strategy, and governance library. This `pilot/` folder is the execution area for proof of concept, simulated runs, manual pilot operations, and evidence capture.

## Structure

- `templates/` contains reusable working templates.
- `scenarios/` contains operational scenario packs.
- `runs/` contains individual pilot runs.
- `reports/` contains pilot summaries and pattern reports.
- `evidence/` contains validation and learning records.

## Operating principle

One user. One real or simulated leadership scenario. One preparation cycle. One rehearsal cycle. One review. One pattern report.

The first goal is to prove the VALOUR loop before building software.
