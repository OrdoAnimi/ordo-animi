# Pilot Intake Template

## User story

As a VALOUR pilot operator, I want to capture the user's role, context, leadership situation, confidence, and desired outcome so that the pilot can start from a clear and usable scenario.

## Acceptance criteria

- User role and context are captured.
- One real or simulated leadership scenario is described.
- Desired outcome is clear.
- Confidence baseline is recorded.
- Sensitive organisational detail is avoided unless required.

## Success criteria

The intake is complete when the operator can select a scenario and write a short situation frame without needing more discovery.
