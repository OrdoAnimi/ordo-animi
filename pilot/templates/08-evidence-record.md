# Evidence Record Template

## User story

As the VALOUR product lead, I want an evidence record for each pilot so that we can judge whether the workflow is useful and repeatable.

## Acceptance criteria

- Baseline confidence is captured.
- After confidence is captured.
- Scenario fit is recorded.
- Usefulness score is recorded.
- Next-step signal is captured.

## Success criteria

The evidence record is successful when it supports a clear decision about the next pilot action.
