# Language Refinement Template

## User story

As a VALOUR pilot user, I want my rough answer refined into clearer versions so that I can speak with the right balance of brevity, authority, empathy, and decision clarity.

## Acceptance criteria

- Original answer is captured.
- Weakness in the answer is identified.
- Short version is produced.
- Executive version is produced.
- Empathetic version is produced.
- Decision version is produced.

## Success criteria

Language refinement is successful when the user can choose at least one improved version they would actually use.
