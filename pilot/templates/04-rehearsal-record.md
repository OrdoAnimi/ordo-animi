# Rehearsal Record Template

## User story

As a VALOUR pilot user, I want to rehearse the difficult part of the conversation so that I can test my clarity, cadence, empathy, authority, and decision framing before the real moment.

## Acceptance criteria

- Rehearsal prompt is recorded.
- User answer is captured.
- Feedback is captured.
- Improved answer is captured.
- At least one development pattern is identified.

## Success criteria

The rehearsal is successful when the user has at least one improved answer they could use in the real conversation.
