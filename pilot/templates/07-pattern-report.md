# Pattern Report Template

## User story

As a VALOUR pilot user, I want a concise pattern report so that I understand my starting challenge, improvement, recurring pattern, and next development focus.

## Acceptance criteria

- Starting challenge is summarised.
- Scenario used is recorded.
- Improvement is identified.
- Primary pattern is named.
- Next focus is recommended.
- Report is short enough to be useful.

## Success criteria

The pattern report is successful when the user can read it quickly and identify one practical behaviour to carry into the next leadership situation.
