# Stage 06 — After-Action Review

## Pilot ID

PILOT-002

## User ID

SIM-002

## What worked

- The Preparation Brief forced the executive translation before rehearsal. This is the highest-value step for an executive-facing scenario — the framing work that happens in Stage 03 directly improves the quality of the rehearsal answer.
- Language Refinement produced four clearly differentiated variants. The Decision version was immediately usable. The Short version was useful as a fallback in case the briefing was cut short.
- The rehearsal question selected ("What is the single biggest risk") was the most likely hard question in an executive briefing of this type. Practising this one question provided broad preparation.

## What did not work

- The first rehearsal answer defaulted entirely to architecture language. The user would have benefited from an explicit prompt in Stage 03 to draft a translated version of their key risk position before rehearsal began.
- The preparation brief needed a more explicit executive audience translation step — a dedicated mode that asks the user to state the business consequence of each key technical position before generating the brief.

## Where clarity improved

Confidence moved from 4 to 7. The biggest single shift was the realisation that executive audiences need the recommendation first and the justification second. The user's instinct was to build to the recommendation — VALOUR reversed that structure.

## What to change next time

1. Add an executive financial framing mode to the Preparation Brief stage. Before rehearsal, the user should translate each key technical position into financial risk, business consequence, and investment outcome language.
2. Add an explicit "recommendation first" prompt to the Rehearsal stage for executive-facing scenarios. The user should be directed to open their answer with the recommendation before explaining the supporting rationale.

## Product gaps identified

1. **Preparation Brief — executive financial framing mode.** The current brief is well-structured but does not have a dedicated mode for translating technical positions into financial and consequence language before rehearsal. This is the most important product improvement for executive-facing scenarios.
2. **Rehearsal stage — executive register prompt.** A short prompt at the start of rehearsal for executive scenarios: "Lead with your recommendation. State it in one sentence. Then support it."

## Operator note

Key improvement: executive-facing preparation brief needs a dedicated financial framing mode that translates technical positions into risk, cost, and business consequence language before rehearsal begins. This is a distinct product need from Pilot 001 (which needed a multi-question structured rehearsal workspace). Both gaps are valid and additive.
