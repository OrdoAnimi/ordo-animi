# Ecosystem Alignment Matrix

## Objective

Bring all ZenCloud / Velocity ecosystem limbs to the same maturity baseline as EA Artefact Generator.

## Reference maturity pattern

EA Artefact Generator is the reference limb because it has clear product intent, ecosystem fit, workspace capability, artefact generation, artefact library, export behaviour, security notes, and local/deployment instructions.

## System map

| System | Role | Current maturity | Alignment target |
|---|---|---:|---|
| Ordo Animi | Brain / command layer | High | Own orchestration, guardrails, decision state, and ecosystem contract |
| PMO Portal | Governed engagement workspace | High | Own artefact lifecycle, governance, visibility, RAID, milestones, handover |
| EA Artefact Generator | Enterprise architecture limb | Reference | Maintain as baseline pattern |
| BA Artefact Generator | Business intent and requirements limb | Medium-high | Complete catalogue, quality panel, traceability panel, handoff flows |
| SA Artefact Generator | Solution architecture limb | Medium | Add lifecycle controls, deployment route, artefact library, PMO handoff |
| PM Artefact Generator | Delivery-management limb | Medium | Add generation engine, persistence, export pipeline, lifecycle controls |

## Required maturity controls

Each limb must have:

- README with product intent and ecosystem role.
- Live or configured deployment route.
- Deployment guide.
- Security and data-handling guide.
- Artefact catalogue.
- Artefact lifecycle mapping.
- PMO Portal handoff model.
- Local run/build instructions.
- Roadmap.
- Licence posture.

## Same-day alignment priority

1. SA: confirm deployment route and add missing lifecycle/output maturity controls.
2. PM: add ecosystem alignment, security/data, and lifecycle/handoff documentation.
3. BA: close MLP gap docs around traceability, quality, and handoff flows.
4. PMI/PMO Portal: confirm it is the lifecycle record of truth and document contracts with all limbs.
5. Ordo Animi: own maturity baseline, alignment matrix, and shared request/response contract.

## Definition of aligned

A limb is aligned when it can answer:

- What professional work does it own?
- What artefacts does it generate?
- What lifecycle states does it support?
- How does it hand off to PMO Portal?
- How does Ordo Animi orchestrate or govern it?
- What data is safe to enter?
- How are outputs exported, reviewed, approved, shared, and locked?
